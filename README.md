# ⚾ Baseball Flash

> 가볍게 즐기는 게임을 한곳에 모아, 골라서 바로 플레이하는 웹 게임 포털

**Baseball Flash**는 별도 설치 없이 브라우저에서 즉시 실행되는 캐주얼 게임 포털입니다. 대표 게임을 크게 노출하는 히어로 영역과 게임 목록 그리드로 구성되어 있으며, 게임을 선택하면 포털 안의 플레이어 슬롯에서 곧바로 실행됩니다.

현재 포털에는 3D KBO 홈런 더비 게임이 수록되어 있습니다.

---

## 🎮 수록 게임

### 홈런 더비 (`homerun-derby.html`)

포수 뒤 중계 시점으로 진행되는 3D 홈런 더비 게임입니다.

- **팀 선택**: KBO 10개 구단 중 선택하면 해당 팀의 홈구장이 자동 설정됩니다.
- **구장 반영**: 잠실·고척 스카이돔·문학·수원·창원·대구·사직·광주·대전 등 실제 구장의 중앙/좌우 담장 거리와 담장 높이, 돔 여부가 반영됩니다.
- **타자 선택**: 팀별 타자 2명 중 선택하며, 타자마다 파워(pw)·정확도(ct) 능력치가 달라 컨택 타이밍 허용치와 타구 위력에 영향을 줍니다.
- **투구 종류**: 직구·슬라이더·커브·체인지업·마구가 가중치에 따라 무작위로 등장하며, 구종별로 속도와 변화(좌우 드리프트, 상하 변화)가 다릅니다.
- **조작**: `스페이스` 또는 `탭`으로 투구를 받고, 다시 눌러 스윙합니다. **안 치고 그냥 보내도 아웃**이므로 적극적으로 노려야 합니다.
- **판정**: 타이밍에 따라 배럴(barrel)·솔리드(solid)·빗맞음(mishit)·헛스윙(whiff)으로 나뉘며, 타구 초속·발사각·좌우 방향이 결정되어 물리 시뮬레이션으로 비거리와 홈런 여부를 계산합니다.
- **게임 규칙**: 아웃 5개를 소진하면 경기가 종료되고, 기록한 홈런 개수와 최장 비거리가 집계됩니다.
- **온라인 랭킹**: Firebase Firestore를 통해 닉네임·팀·타자·홈런 수·최장 비거리를 실시간 랭킹으로 등록·조회할 수 있습니다.
- **사운드**: Web Audio API로 타격음·홈런·헛스윙 효과음을 생성하며 음소거 토글을 지원합니다.

## 🗂 프로젝트 구조

```
baseball-flash/
├── index.html          # 포털 셸 (히어로 + 게임 그리드 + 플레이어 슬롯)
├── portal.js           # 매니페스트 기반으로 포털 UI 구성 및 게임 실행 관리 (ES 모듈)
├── games.js            # 게임 목록(매니페스트) 정의
├── styles.css          # 포털 스타일
└── homerun-derby.html  # 홈런 더비 게임 본체 (단일 HTML)
```

## 🛠 기술 스택

| 구분 | 사용 기술 |
| --- | --- |
| 프론트엔드 | HTML, CSS, JavaScript (ES 모듈) |
| 3D 렌더링 | [three.js](https://threejs.org/) r128 |
| 온라인 랭킹 | Firebase App / Firestore (compat 10.12.2) |
| 오디오 | Web Audio API |

외부 라이브러리는 CDN(cdnjs, gstatic)에서 로드합니다.
